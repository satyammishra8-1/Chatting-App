QuickChat

A modern real-time communication and productivity platform built with React, Node.js, MongoDB, and Socket.IO.
QuickChat goes beyond traditional messaging applications by combining secure communication, productivity tools,
multilingual support, reminders, scheduling, and personal knowledge management into a single platform.

---

## Features

### Real-Time Messaging

* Instant one-to-one messaging
* Real-time communication using Socket.IO
* Message delivery and read status
* Message reactions
* Reply to messages

### 🔒 Secure Messaging

* PIN-protected secure messages
* Email OTP verification for PIN updates
* Secure message unlocking system
* Privacy-focused communication

### 🌐 Multilingual Communication

* Automatic message translation
* User preferred language support
* Improved accessibility for diverse users

### ⏰ Productivity Features

* Reminder creation and notifications
* Scheduled message delivery
* Personal task and note management
* Link organization
* Idea tracking

### 🎤 Smart Communication

* Voice-to-text input
* Image sharing
* Profile customization
* Online user status

### 👤 User Management

* JWT Authentication
* Email OTP Verification
* Profile Picture Upload
* Language Preferences
* Account Personalization

## 🏗️ System Architecture
Frontend → React.js
Backend → Node.js + Express.js
Database → MongoDB Atlas
Real-Time Engine → Socket.IO
Authentication → JWT
Email Service → Nodemailer
Deployment → Vercel + Render
